# Descriptions des cartes — Programme R & Les Contractuels

> Format : **Nom** — description à coller dans le champ `Description` du CardData.
> Pour les unités : passif positionnel + keyword. Pour les sorts : effet.
> Le `-` indique qu'il n'y a rien à mettre (la carte n'a ni keyword ni passif).
