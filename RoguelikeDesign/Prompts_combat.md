## Notes de génération

### Style cohérence
Tous les assets partagent : `2D flat cartoon illustration` / `retro corporate lo-fi` / `physical imperfect handmade feel`  
Éviter le rendu 3D photo-réaliste — rester dans l'illustration plate légèrement texturée.

### Backgrounds transparents
- **Midjourney :** générer sur fond blanc, puis retirer le fond sous Photopea (Calque > Teinte/Saturation ou baguette magique)
- **Leonardo AI :** activer "Remove Background" ou "Transparent Background" dans les options si disponible avec Phoenix 1.0

### Dimensions & intégration Unity
Exporter en PNG avec transparence. Les tailles cibles ci-dessus sont indicatives — Unity adapte via RectTransform, donc une marge de ±20% ne pose pas de problème.

### Ordre de génération recommandé
1. Fond Tableau (donne le ton pour tout le reste)
2. Ardoise Plateau (élément central le plus complexe)
3. Post-its + Pochette (même famille stylistique)
4. Banderole + Cercles Score
5. Scotch Carte en dernier (le plus simple)
